## Bug 2 — Student Update Not Saved

**Summary:** Student update not saved to database

**Steps to Reproduce:**
1. Select existing student
2. Edit student data
3. Click Save

**Expected Result:**
- Student data updated in database

**Actual Result:**
- Old data remains unchanged

**Severity:** High

**Status:** Open
